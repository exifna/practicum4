# Online game "КБ ПУМ"

## Installition guide

```shell
# Download project
git clone https://github.com/exifna/practicum4  # copy project to local folder
cd practicum4  # go to project folder
# Create venv
python -m venv venv  # create virtual environment
source venv/bin/activate # if you on UNIX system
.\venv\Scripts\activate  # if on Windows system 
 # Installing dependencies
 pip install -r requirements.txt
 # Run api
 cd server   
 python api.py
 # Run client
 # In new terminal/console 
 cd client
 python main.py
```
